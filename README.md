# Customer Purchase Behavior Analysis

## Project Overview

This project analyzes customer shopping behavior using transactional data from 3,900 purchases across multiple product categories. The objective is to identify spending patterns, customer segments, product preferences, and subscription behavior to support data-driven business decisions.

## Dataset Summary

* Total Records: 3,900
* Total Features: 18
* Data Includes:

  * Customer Demographics (Age, Gender, Location)
  * Purchase Information (Product, Category, Purchase Amount)
  * Subscription Status
  * Discounts and Promotions
  * Shipping Preferences
  * Customer Reviews and Ratings
  * Purchase History

## Tools & Technologies

* Python (Pandas, NumPy)
* PostgreSQL
* Power BI
* Jupyter Notebook
* SQL

## Data Cleaning & Preparation

* Performed exploratory data analysis using Python.
* Handled missing values in the Review Rating column using median imputation by category.
* Standardized column names for consistency.
* Created age group categories for customer segmentation.
* Validated data quality and removed redundant fields.
* Loaded cleaned data into PostgreSQL for SQL analysis.

## Business Questions Answered

1. What is the total revenue generated by each gender?
2. Which customers spend above average while using discounts?
3. What are the top-rated products?
4. How does purchase behavior differ across shipping types?
5. How do subscribers compare with non-subscribers?
6. Which products are most dependent on discounts?
7. How are customers segmented into New, Returning, and Loyal groups?
8. What are the top purchased products within each category?
9. Are repeat buyers more likely to subscribe?
10. Which age groups contribute the most revenue?

## Dashboard Features

* Revenue Analysis

* Product Performance Analysis
* Subscription Analysis
* Review Rating Analysis

## Key Insights

* Young Adult customers contributed the highest revenue.
* Customer purchasing behavior varies across demographic groups.
* Subscription status influences customer spending patterns.
* Certain products show higher dependency on discounts.
* Customer loyalty plays a significant role in revenue generation.

## Business Recommendations

* Increase subscription adoption through exclusive offers.
* Develop loyalty programs for repeat customers.
* Optimize discount strategies to improve profitability.
* Focus marketing efforts on high-value customer segments.
* Promote top-performing and highly rated products.
